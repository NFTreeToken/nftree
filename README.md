# NFTree

## Frontend Features

- TypeScript
- Material-UI

## Requirements

- Latest Node
- Yarn
- Ganache: https://www.trufflesuite.com/ganache

## Commands

```
yarn contracts:compile
yarn contracts:deploy
yarn contracts:test

yarn frontend:start
yarn frontend:typecheck
```
